# Overview

As a Software engineer learning and exploring modern development and applications I was drawn to learn about Kotlin. I've had an idea for an Artificial Intelligence structure while I was learning data science tools at my TripleTen bootcamp. Kotlin provided a perfect frontend application for that structure so I generated this code as a way to explore the feasibility of this design.
The basic functionality of this feasibility test is to take a message from the user and act with it in a predictable, consistent way. Beginning with a simple notes taking echo program the expectation is to have the message and conversation be processed and returned with a response from the python Artificial Consciousness Intelligence & Interface (ACI).

## Purpose

Determine realistic usability and mastery of Kotlin Syntax while constructing production-ready Android UI patterns with Jetpack Compose. Limiting conversations to 100 messages in standard chat UI, with overflow removal. The Debug Panel tracks Users and ACI message numbers showcasing clean state management.
