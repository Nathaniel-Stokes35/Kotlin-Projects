# Overview

This project is my first program written using the Kotlin programming language and the IntelliJ IDEA development environment.

The software is a simple console application that prints the phrase:

"Hello World! - We are in the Kotlin Program!"

to the terminal.

The purpose of creating this program is to familiarize myself with the Kotlin language as I continue developing my skills in software engineering. I am particularly interested in learning Kotlin because it is commonly used for mobile application development, and I plan to eventually create a Kotlin-based front-end application for my senior project.

[Software Demo Video](https://youtu.be/dMdXa3P5N8I)

# Development Environment

The software was developed using the following tools:

- IntelliJ IDEA 2025.3.3
- Git and GitHub for version control
- A personal desktop computer running Windows with an AMD Ryzen 9 3900X CPU and AMD Radeon RX 6800 GPU

The programming language used in this project is Kotlin. Although this is my first Kotlin program, Java was the first programming language I learned in high school in 2011, which helped me understand many of the concepts used in Kotlin.
